## 在windows cmd中设置代理
>set http_proxy=http://username:password@ip:port
这样在cmd中的所有网络请求都会通过代理访问网络

使用easy_install 安装软件
>easy_install.exe -i http://mirrors.aliyun.com/pypi/simple/ django

而使用pip时,指定mirror
>pip install flask -i http://pypi.v2ex.com/simple
